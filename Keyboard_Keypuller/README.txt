# Summary

----

Check out McGlovin's remix, it's thinner and should fit more keyboards

http://www.thingiverse.com/thing:2222657

----

This a keypuller for mechanical keyboards, I made it as simple as possible to print.

The .step file is also available in case anyone wants to customize it. Send me a message if you need another CAD format.

# Print Settings

Printer Brand: MakerBot
Printer: MakerBot Replicator 2
Rafts: Yes
Supports: No
Resolution: .25
Infill: 45%

Notes: 
The bottom "claws" were made thin enough so that there'd be no infill, that's so it can offer less resistance when pushing down.
